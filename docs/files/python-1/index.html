Python 1 Programming Course Outline

PCEP-30-02 Certified Entry-Level Python Programmer Aligned August 12 - December 18

Course Overview

This course introduces students to programming fundamentals using Python while preparing them for the PCEP-30-02 (Certified Entry-Level Python Programmer) certification exam. The curriculum follows the official PCEP exam syllabus, ensuring students master both practical programming skills and certification requirements. The course is structured into 8 units, with the first 4 units directly aligned with the four PCEP exam sections.

Unit 1: Computer Programming and Python Fundamentals

August 12 - 22 2 weeks - PCEP Section 1 (18% of exam)

PCEP-30-02 Objectives Covered:

PCEP 1.1 – Understand fundamental terms and definitions

  • interpreting and the interpreter, compilation and the compiler
  • lexis, syntax, and semantics

PCEP 1.2 – Understand Python’s logic and structure

  • keywords, instructions, indentation, comments

PCEP 1.3 – Introduce literals and variables into code and use different numeral systems

  • Boolean, integers, floating-point numbers, scientific notation, strings
  • binary, octal, decimal, and hexadecimal numeral systems
  • variables, naming conventions, implementing PEP-8 recommendations

PCEP 1.4 – Choose operators and data types adequate to the problem

  • numeric operators: ** * / % // + –
  • string operators: * +, assignments and shortcut operators
  • operators: unary and binary, priorities and binding
  • bitwise operators: ~ & ^ | « », Boolean operators: not and or
  • Boolean expressions, relational operators ( == != > >= < <= )
  • the accuracy of floating-point numbers, type casting

PCEP 1.5 – Perform Input/Output console operations

  • print() and input() functions
  • sep and end keyword parameters of the print() function

Topics:

  • What is programming and computational thinking
  • Python’s place in programming languages
  • Installation and development environment setup
  • Python syntax, keywords, and structure
  • Data types and literals (all numeral systems)
  • Variables and PEP-8 naming conventions
  • All operator types and precedence
  • Basic input/output operations
  • Unit Assessment: PCEP Section 1 aligned practical project

Unit 2: Control Flow - Conditional Blocks and Loops

2 weeks - PCEP Section 2 (29% of exam)

PCEP-30-02 Objectives Covered:

PCEP 2.1 – Make decisions and branch the flow with the if instruction

  • conditional statements: if, if-else, if-elif, if-elif-else
  • multiple conditional statements
  • nesting conditional statements

PCEP 2.2 – Perform different types of iterations

  • the pass instruction
  • building loops with while, for, range(), and in
  • iterating through sequences
  • expanding loops with while-else and for-else
  • nesting loops and conditional statements
  • controlling loop execution with break and continue

Topics:

  • Boolean logic and expressions
  • Conditional execution with if statements
  • Complex conditional structures (elif, nested conditions)
  • while loops and loop control
  • for loops with range() and sequences
  • Loop nesting and best practices
  • break, continue, and else clauses in loops
  • Unit Assessment: Decision-making and iteration project

Unit 3: Data Collections - Lists, Tuples, Dictionaries, and Strings

2 weeks - PCEP Section 3 (25% of exam)

PCEP-30-02 Objectives Covered:

PCEP 3.1 – Collect and process data using lists

  • constructing vectors, indexing and slicing, the len() function
  • list methods: append(), insert(), index(), etc.
  • functions: len(), sorted()
  • the del instruction, iterating through lists with the for loop
  • initializing loops, the in and not in operators
  • list comprehensions, copying and cloning, lists in lists

PCEP 3.2 – Collect and process data using tuples

  • tuples: indexing, slicing, building, immutability
  • tuples vs. lists: similarities and differences
  • lists inside tuples and tuples inside lists

PCEP 3.3 – Collect and process data using dictionaries

  • dictionaries: building, indexing, adding and removing keys
  • iterating through dictionaries and their keys and values
  • checking the existence of keys
  • keys(), items() and values() methods

PCEP 3.4 – Operate with strings

  • constructing strings, indexing, slicing, immutability
  • escaping using the \ character
  • quotes and apostrophes inside strings, multi-line strings
  • basic string functions and methods

Topics:

  • Lists: creation, manipulation, and methods
  • List comprehensions and advanced techniques
  • Tuples: immutable sequences and their uses
  • Dictionaries: key-value pairs and methods
  • String operations and formatting
  • Choosing appropriate data structures
  • Unit Assessment: Data processing and analysis project

Unit 4: Functions and Exceptions

2 weeks - PCEP Section 4 (28% of exam)

PCEP-30-02 Objectives Covered:

PCEP 4.1 – Decompose the code using functions

  • defining and invoking user-defined functions and generators
  • the return keyword, returning results
  • the None keyword, recursion

PCEP 4.2 – Organize interaction between the function and its environment

  • parameters vs. arguments
  • positional, keyword and mixed argument passing
  • default parameter values
  • name scopes, name hiding (shadowing)
  • the global keyword

PCEP 4.3 – Python Built-In Exceptions Hierarchy

  • BaseException, Exception, SystemExit, KeyboardInterrupt
  • abstract exceptions, ArithmeticError, LookupError
  • IndexError and KeyError, TypeError and ValueError exceptions
  • the AssertError exception along with the assert keyword

PCEP 4.4 – Basics of Python Exception Handling

  • try-except, try-except Exception
  • ordering the except branches
  • propagating exceptions through function boundaries
  • delegating responsibility for handling exceptions

Topics:

  • Function definition and invocation
  • Parameters, arguments, and return values
  • Variable scope and the global keyword
  • Built-in vs. user-defined functions
  • Exception hierarchy and common exceptions
  • Exception handling with try-except blocks
  • Unit Assessment: Function-based application with error handling

Unit 5: Advanced Python Programming Concepts

1 week

Topics:

  • File operations (reading, writing, file modes)
  • Working with CSV and text files
  • Regular expressions basics
  • Date and time handling
  • Introduction to modules and packages
  • Python Standard Library overview
  • Mini-Project: File processing application

Unit 6: Object-Oriented Programming Fundamentals

1 week

Topics:

  • Introduction to OOP concepts
  • Classes and objects
  • Attributes and methods
  • Constructors and destructors
  • Inheritance basics
  • Encapsulation principles
  • Mini-Project: Simple class-based application

Unit 7: Python for Data Analysis and Visualization

1 week

Topics:

  • Introduction to NumPy and Pandas
  • Data manipulation and cleaning
  • Basic statistical operations
  • Data visualization with Matplotlib
  • Reading data from various sources
  • Introduction to Jupyter notebooks
  • Mini-Project: Data analysis dashboard

Unit 8: Real-World Python Applications and Career Preparation

1 week

Topics:

  • Web scraping with requests and BeautifulSoup
  • API interaction and JSON processing
  • Automation and scripting
  • Introduction to web development (Flask basics)
  • Version control with Git
  • Python career paths and next steps
  • Final Project: Comprehensive application addressing real-world problem
  • PCEP Exam Preparation: Practice tests and review

PCEP-30-02 Exam Preparation Integration

Throughout the Course:

  • Daily PCEP Practice: Quick review questions aligned with exam objectives
  • Weekly PCEP Quizzes: Section-specific practice tests
  • Exam Simulation: Full-length practice exams in Units 4, 6, and 8
  • Study Groups: Peer review sessions for PCEP concepts

PCEP Exam Readiness Checklist:

  • ✅ Master all four PCEP sections (Units 1-4)
  • ✅ Complete section-specific practice tests
  • ✅ Demonstrate proficiency in hands-on programming
  • ✅ Pass mock PCEP examinations
  • ✅ Understand exam format and timing
