Assignment 3.3 – Looping Through Lists

Objective

Practice using for loops and membership checks with lists.

Part A: Basic Loops

  1. Write a program that creates a list of 5 favorite foods and prints them one by one.
  2. Create a list of numbers [2, 4, 6, 8, 10]. Use a loop to print each number doubled.

Part B: Using Indexes

  1. Given this list: 
    animals = ["cat", "dog", "fish"]
    Write a loop that prints: 
    0 cat
1 dog
2 fish

Part C: Membership

  1. Make a list of colors. Ask the user to type a color. If it’s in the list, print “Found it!”, otherwise print “Not here.”
  2. Why might checking membership be useful in a game?

Part D: Mini Challenge

Write a program that:

  • Creates a list of numbers from 1 to 20
  • Loops through them
  • Prints only the multiples of 3
