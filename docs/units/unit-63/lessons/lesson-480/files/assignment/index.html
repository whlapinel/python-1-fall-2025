Assignment 3.5 – Dictionary Basics

Objective

Learn how to create, access, add, and remove items from dictionaries.

Part A: Creating and Accessing

  1. Create a dictionary called car with keys: make, model, year.

    • Print the model.

  2. Add a new key color with any value.

    • Print the full dictionary.

Part B: Modifying and Removing

  1. Start with this dictionary: 
    movie = {"title": "Inception", "year": 2010}
    • Change the year to 2012
    • Add a key rating with a value of 9
    • Remove the title key
    • Print the result

Part C: Membership

  1. Create a dictionary student with keys: name, age, grade.
  2. Write code that checks if age is in the dictionary. If yes, print the value.

Part D: Mini Challenge

Write a program that:

  • Creates a dictionary for a song with keys: title, artist, length
  • Adds a new key album
  • Prints just the artist
  • Removes the length
