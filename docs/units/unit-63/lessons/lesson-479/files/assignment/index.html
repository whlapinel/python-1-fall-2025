Assignment 3.4 – Working with Tuples

Objective

Understand how tuples work and practice using packing, unpacking, and immutability.

Part A: Basics

  1. Create a tuple with 3 numbers. Print the second number.
  2. Try to change one element of the tuple. What happens?

Part B: Packing & Unpacking

  1. Pack your first name and last name into a tuple called person.
  2. Unpack person into two variables and print them.

### Part C: Returning Tuples 1. Write a function first_and_last(lst) that returns a tuple with the first and last elements of a list. 2. Call the function on [10, 20, 30, 40] and unpack the result into two variables.

Part D: Mini Challenge

Write a program that:

  • Creates a tuple (100, 200, 300)
  • Unpacks into x, y, z
  • Prints the sum of all three values
