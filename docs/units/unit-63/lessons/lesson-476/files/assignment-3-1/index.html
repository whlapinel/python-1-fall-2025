Assignment 3.1 (Prepare)

Instructions

Complete the following exercises to practice working with Python lists. Write your answers in a .py file or in your notebook.

Part A: Create and Access

  1. Create a list called animals with at least 4 animals of your choice.
  2. Print the first and last items in the list.
  3. Print the length of the list using len().

Part B: Modify

  1. Change the 2nd item in your list to a different animal.
  2. Add a new animal to the end of the list with .append().
  3. Remove one animal by its name using .remove().

Part C: Apply

  1. Create a list of 5 numbers.
  2. Print the 3rd number in the list.
  3. Replace the last number with 100.
  4. Add a new number at the end.
  5. Print the updated list.

Submission

Turn in your completed code file for review. Be sure your file runs without errors.

Assignment 3.1 Goals

  • Practice creating and modifying lists
  • Learn to access items by index
  • Use basic list methods (append, remove, pop)
