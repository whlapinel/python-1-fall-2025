Assignment 3.6 – Dictionary Iteration & Methods

Objective

Practice looping through dictionaries and using built-in dictionary methods.

Part A: Iteration

  1. Create a dictionary states with 3 states and their capitals.
    • Loop through keys and print them.
    • Loop through values and print them.
    • Loop through both key and value to print State -> Capital.

Part B: get()

  1. Given: 
    student = {"name": "Bob", "grade": "11th"}
    • Use get() to safely print the student’s age with a default message.

Part C: update()

  1. Start with: 
    car = {"make": "Toyota", "model": "Camry"}
    • Use update() to add the year and change the model.
    • Print the dictionary.

Part D: del

  1. Create a dictionary book with keys: title, author, year.
  2. Delete the year key and print the result.

Part E: Mini Challenge

Write a program that:

  • Creates a dictionary of 3 students and their test scores
  • Loops through and prints each student with their score
  • Updates one student’s score using update()
  • Deletes one student from the dictionary
