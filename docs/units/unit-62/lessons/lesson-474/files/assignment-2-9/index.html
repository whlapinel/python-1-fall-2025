Assignment 2.9

Supporting Resources

Goal: More practice with if statements, while loops, and for loops.

Problem 1

Write a program that asks the user for a number and prints “Even” if it is divisible by 2, otherwise prints “Odd”.

Problem 2

Write a program that prints the numbers 5 through 15 using a for loop.

Problem 3

Write a program that counts down from 10 to 1 using a while loop, then prints “Done!”.

Problem 4

Write a program that asks the user for their age. If 18 or older, print “Adult”. Else print “Minor”.

Problem 5

Write a program that prints the multiples of 4 from 4 to 40 using a for loop.

Problem 6

Write a program that asks the user to enter numbers until they type 0. Print “Finished” when the loop ends.

Problem 7

Write a program that prints all odd numbers from 1 to 19 using a for loop.

Problem 8

Write a program that asks the user for a number. If it is greater than 100, print “Big”. If it is between 50 and 100, print “Medium”. Otherwise, print “Small”.

Problem 9

Write a program that asks the user for a word. Use a for loop to print each character on its own line.

Problem 10

Write a program that starts with x = 1 and uses a while loop to print numbers up to 20. Use continue to skip multiples of 5.
