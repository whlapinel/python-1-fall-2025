Assignment 1.3:

Unit 1, Lesson 1 - PREPARE Assignment Due: Next class period

Setup Instructions

  1. Open your Python interpreter (type python in Command Prompt/Terminal)
  2. You should see the >>> prompt
  3. Type each program exactly as shown
  4. Take screenshots or write down what happens

Program 1: Personal Introduction (15 points)

Type this program and run it:

print("Hi there!")
print("My name is [YOUR ACTUAL NAME]")
print("I am learning Python programming")

Replace [YOUR ACTUAL NAME] with your real name

What to submit:

  • Write down exactly what your program printed
  • Screenshot optional but encouraged

Program 2: Python Calculator (20 points)

Try each calculation and write down the result:

8 + 5

20 - 7

6 * 9  

15 / 3

2 ** 4

Program 3: Variables Practice (20 points)

Type and run this program:

my_favorite_number = 7
my_age = 16
print("My favorite number is", my_favorite_number)
print("I am", my_age, "years old")

Program 4: Interactive Program (25 points)

Type this program exactly:

print("Welcome to my survey!")
favorite_color = input("What is your favorite color? ")
favorite_food = input("What is your favorite food? ")
print("Cool! You like", favorite_color, "and", favorite_food)
print("Thanks for answering!")

Program 5: Create Your Own (20 points)

Now write your OWN program that:

  1. Prints a welcome message
  2. Asks the user for TWO pieces of information
  3. Does ONE math calculation (like adding two numbers)
  4. Prints everything in a final message
