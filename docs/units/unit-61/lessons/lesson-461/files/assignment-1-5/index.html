Assignment 1.5 – Boolean Logic

Goal: Practice writing and testing boolean expressions.

Instructions:

  1. Write a Python program called boolean_quiz.py.
  2. Your program should:
    • Ask the user to enter two numbers (use input() and int() to convert).
    • Print out the results of the following checks:
      1. Is the first number greater than the second?
      2. Are the two numbers equal?
      3. Is at least one number greater than 10?
      4. Is the first number NOT zero?
    • Each result should print as True or False.

Example Run:

Enter the first number: 5
Enter the second number: 12
Is the first number greater than the second? False
Are the numbers equal? False
Is at least one number greater than 10? True
Is the first number not zero? True

Submission: Upload your .py file when finished.
