Assignment 1.8: Unit 1 Assessment - Interactive Python Console Application

Type: PERFORM | Due: End of Week 2 | Points: 100

Overview

Create a comprehensive interactive console application that demonstrates mastery of all PCEP Section 1 objectives. Your program should be engaging, user-friendly, and showcase proper Python programming practices.

PCEP Section 1 Objectives Demonstrated

Your project must clearly demonstrate ALL of the following PCEP-30-02 Section 1 objectives:

PCEP 1.1 - Fundamental Terms and Definitions

  • Use Python as an interpreted language (no compilation step needed)
  • Demonstrate understanding of proper Python syntax and semantics
  • Include comments explaining your code’s lexical elements

PCEP 1.2 - Python Logic and Structure

  • Use Python keywords appropriately
  • Implement proper indentation for code blocks
  • Include meaningful comments throughout your code

PCEP 1.3 - Literals and Variables

  • Use Boolean, integer, floating-point, and string literals
  • Demonstrate at least 2 different numeral systems (binary, octal, hex)
  • Create variables following PEP-8 naming conventions
  • Use scientific notation where appropriate

PCEP 1.4 - Operators and Data Types

  • Use arithmetic operators: **, *, /, %, //, +, −
  • Use string operators: *, +
  • Use assignment and shortcut operators
  • Demonstrate operator precedence understanding
  • Use Boolean operators: not, and, or
  • Use relational operators: ==, !=, >, >=, <, <=
  • Include type casting operations

PCEP 1.5 - Input/Output Operations

  • Use print() function with sep and end parameters
  • Use input() function for user interaction
  • Combine input with type conversion

Project Options

Choose ONE of the following project types:

Option A: Personal Information Calculator

Create an interactive program that:

  • Collects personal information (name, birth year, height, etc.)
  • Performs calculations (age, height conversions, etc.)
  • Uses different numeral systems to display interesting facts
  • Formats output professionally using print() parameters

Option B: Number System Converter and Calculator

Create a program that:

  • Accepts numbers in different bases (binary, octal, decimal, hex)
  • Converts between numeral systems
  • Performs calculations with the converted numbers
  • Shows results in multiple formats
  • Includes interactive menu system

Option C: Interactive Quiz Game

Create a program that:

  • Asks users questions about Python concepts
  • Accepts and evaluates user answers
  • Keeps score using various operators
  • Provides feedback with formatted output
  • Uses Boolean logic for answer checking

Option D: Custom Project (Instructor Approval Required)

Design your own project that demonstrates all PCEP Section 1 objectives

  • Submit a brief proposal before starting
  • Must be approved by instructor
  • Should be of similar complexity to other options

Technical Requirements

Code Quality (25 points)

  • PEP-8 Compliance: Proper variable naming, spacing, line length
  • Comments: Meaningful comments explaining complex logic
  • Structure: Well-organized code with logical flow
  • Error Prevention: Handle potential user input errors gracefully

PCEP Objective Coverage (50 points)

  • Complete Coverage: All 5 PCEP Section 1 objectives clearly demonstrated
  • Correct Implementation: Proper use of all required language features
  • Appropriate Usage: Choose right operators/data types for each task
  • Understanding: Code shows deep understanding, not just surface usage

User Experience (15 points)

  • Clear Instructions: Users understand what to do
  • Professional Output: Well-formatted, easy-to-read results
  • Interactive Design: Engaging user interaction throughout
  • Robust Input Handling: Program doesn’t crash on unexpected input

Documentation (10 points)

  • Header Comment: Program purpose, author, date
  • Function Documentation: If you create any functions (not required)
  • Inline Comments: Explain complex calculations or logic
  • README: Brief explanation of how to run your program
