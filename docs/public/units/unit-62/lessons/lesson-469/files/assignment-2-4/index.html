Assignment 2.4 – Prepare

Goal: Practice writing and controlling while loops.

Part 1: Predict the Output

x = 1
while x < 4:
    print(x)
    x = x + 1
print("Done")

Part 2: Debugging

Fix this code so it prints numbers 1 through 5:

count = 1
while count <= 5:
print(count)
count = count + 1

Part 3: Write Your Own

  1. Ask the user for a number.
  • While the number is less than 10, keep asking again.
  • When they finally enter 10 or more, print “Thanks!”
  1. Write a guessing game.
  • Set a secret number (like 7).
  • Keep asking the user to guess until they get it right.
  • Print “Correct!” when they do.
  1. Multiplication practice.
  • Ask the user a multiplication problem, like 3 * 4.
  • While they give the wrong answer, keep asking.
  • When they get it right, print “Well done!”
  1. Countdown timer.
  • Ask the user for a number n.
  • Use a while loop to count down from n to 1.
  • Print “Blast off!” at the end.
  1. Collecting names.
  • Keep asking the user to enter a name.
  • Stop only when they type “stop”.
  • After the loop ends, print all the names they entered.
