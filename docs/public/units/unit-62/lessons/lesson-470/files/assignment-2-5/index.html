Assignment 2.5 – Prepare

Goal: Practice nesting, while–else, break, and continue.

Part 1: Predict the Output

x = 3
while x > 0:
    if x % 2 == 0:
        print("even")
    else:
        print("odd")
    x = x - 1
else:
    print("done")

Part 2: Debugging

Fix this code so it stops correctly:

while True:
    name = input("Enter a name (or 'stop' to end): ")
    if name = "stop":
        break
    print("Hello", name)

Part 3: Write Your Own

  1. Password checker
  • Ask the user for a password until they type “letmein”
  • If they get it right, print “Welcome” and stop
  1. Skip multiples of 3
  • Use a loop from 1 to 10
  • Skip printing numbers that are multiples of 3 (use continue)
  1. Countdown with else
  • Start at 5 and count down to 1
  • Use while … else to print “Blast off!” when finished
  1. Nested logic
  • Keep asking the user for a number
  • If it’s even, print “even”
  • If it’s odd, print “odd”
  • Stop the loop if they enter a negative number
  1. Number collector
  • Keep asking the user for numbers until they type 0
  • Store all positive numbers in a list
  • After the loop, print the list
