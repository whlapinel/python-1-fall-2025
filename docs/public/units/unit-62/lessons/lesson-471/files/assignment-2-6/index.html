Assignment 2.6 – Prepare

Goal: Practice writing for loops with ranges and sequences.

Part 1: Predict the Output

for i in range(3):
    print("Hello")

Part 2: Debugging

Fix this code so it prints the numbers 1 to 5:

for i in range(1, 5):
print(i)

Part 3: Write Your Own

  1. Write a for loop that prints the numbers 1 through 10.

  2. Write a for loop that prints the letters of your name, one per line.

  3. Write a for loop that prints the squares of the numbers 1 through 5.

  4. Write a program that asks the user for a word and prints each character followed by !.

  5. Write a program that prints only the even numbers from 2 through 20 using for and range().
