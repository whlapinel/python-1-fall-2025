Python 1 – Unit 3: Data Structures (2 Weeks)

Scope

  • Lists: creation, indexing, slicing, methods
  • Tuples: immutability, unpacking
  • Dictionaries: key-value pairs, iteration, lookups
  • Basic operations: membership, iteration, adding/removing elements
  • Real-world applications and practice problems

Daily Breakdown

Week 1

Day 1 (Lesson 3.1) – Lists: Basics

  • Creating lists, indexing, modifying elements.
  • Assignment 3.1: simple practice with creating/modifying lists.

Day 2 (Lesson 3.2) – Lists: Slicing & Methods

  • Slicing, append, insert, remove, pop, len.
  • Assignment 3.2: list slicing challenges.

Day 3 (Lesson 3.3) – Lists: Iteration

  • For loops with lists, using in.
  • Assignment 3.3: loop through a list to solve mini problems.

Day 4 (Lesson 3.4) – Tuples

  • Tuples vs lists, immutability, packing/unpacking.
  • Assignment 3.4: practice tuple unpacking and swaps.

Day 5 (Rehearse 3.1) – Quiz / Paper

  • Assess lists & tuples.

Week 2

Day 6 (Lesson 3.5) – Dictionaries: Basics

  • Key/value pairs, creating dictionaries, accessing values.
  • Assignment 3.5: dictionary practice problems.

Day 7 (Lesson 3.6) – Dictionaries: Iteration & Methods

  • Looping over keys/values/items, get, update, del.
  • Assignment 3.6: write programs using dicts for lookups.

Day 8 (Lesson 3.7) – Mixed Practice

  • Integrating lists, tuples, and dicts in the same problem.
  • Assignment 3.7: small scenario project (ex: phone book or class roster).

Day 9 (Rehearse 3.2) – Paper Quiz

  • Short written problems on lists, tuples, dicts.

Day 10 (Perform 3.1) – Unit Exam

  • Paper exam covering lists, tuples, and dictionaries.

Assignment Structure

  • Prepare: 3–4 per week → Assignments 3.1, 3.2, 3.3, 3.4 (Week 1), Assignments 3.5, 3.6 (Week 2).
  • Rehearse: Quiz 3.1 (end of Week 1), Quiz 3.2 (Day 9).
  • Perform: Exam 3.1 (Day 10).
