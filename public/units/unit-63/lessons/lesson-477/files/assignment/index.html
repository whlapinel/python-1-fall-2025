Assignment 3.2 – List Slicing and Methods

Objective

Practice creating slices of lists and using common list methods.

Part A: Slicing

  1. Given the list:

    nums = [2, 4, 6, 8, 10, 12, 14]
    • Print the first 3 numbers.
    • Print everything from the 3rd element to the end.
    • Print the middle 3 numbers.

  2. What does this print?

    letters = ["a", "b", "c", "d", "e", "f"]
print(letters[1:5])

Part B: Methods

  1. Start with the list:

    fruits = ["apple", "banana", "cherry"]

    Do the following:

    • Append “orange”
    • Insert “grape” at index 1
    • Remove “banana”
    • Pop the last element
    • Print the final list

  2. Write code that:

    • Creates an empty list colors
    • Appends three colors of your choice
    • Prints the length of the list using len()

Part C: Mini Challenge

Write a program that:

  • Starts with [100, 200, 300]
  • Inserts 50 at the beginning
  • Appends 400
  • Removes 200
  • Prints the final list
