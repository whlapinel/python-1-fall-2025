Assignment 2.7 – Prepare

Goal: Practice different for loop patterns without using lists.

Part 1: Predict the Output

for i in range(1, 10, 3):
    print(i)

Part 2: Debugging

Fix this code so it prints the numbers 1 through 5:

for i in range(1, 5):
print(i)

Part 3: Write Your Own

  1. Write a for loop that prints the numbers from 10 down to 1.

  2. Write a for loop that prints the multiples of 5 from 5 to 50.

  3. Write a for loop that prints each character of the word “Python” on its own line.

  4. Write a for loop that prints the squares of the numbers 1 through 10.

  5. Write a program that prints only the odd numbers between 1 and 20.
