Assignment 2.8 – Prepare (Review)

Goal: Practice combining while loops, for loops, and if statements. Approx. 10 problems.

Problem 1

Write a program that prints the numbers 1 through 10 using a for loop.

Problem 2

Write a program that prints the numbers 10 down to 1 using a while loop.

Problem 3

Write a program that asks the user for a number and prints whether it is even or odd using an if statement.

Problem 4

Write a for loop that prints the multiples of 3 from 3 to 30.

Problem 5

Write a program that uses a while loop to keep asking the user for a password until they type secret.

Problem 6

Write a for loop that prints the squares of the numbers 1 through 5.

Problem 7

Write a program that asks the user for a number.

  • If it’s greater than 100, print “Big”.
  • Else if it’s at least 50, print “Medium”.
  • Else, print “Small”.

Problem 8

Write a program with a while loop that counts down from 5 to 1, then prints “Blast off!”.

Problem 9

Write a program that prints all odd numbers from 1 to 20 using a for loop.
